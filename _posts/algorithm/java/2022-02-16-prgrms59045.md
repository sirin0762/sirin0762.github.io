# 문제
[프로그래머스 보호소에서 중성화한 동물 링크](https://programmers.co.kr/learn/courses/30/lessons/59045)
# 내 생각
2개의 테이블을 조인 한 후 특정 조건을 확인하면 된다.

# 내 코드
```mysql
SELECT
    ai.ANIMAL_ID,
    ai.ANIMAL_TYPE,
    ai.NAME
FROM
    ANIMAL_INS AS ai
JOIN
    ANIMAL_OUTS AS ao
    ON ai.ANIMAL_ID = ao.ANIMAL_ID
WHERE
    ai.SEX_UPON_INTAKE != ao.SEX_UPON_OUTCOME;

```
